CSS Variables

To change all heading colors, set the --heading-color variable to the desired color. For example, to set all heading colors to the primary color as follows:

--heading-color: var(--primary);

CSS Classes

Individual heading colors can also be set using the primary, secondary, tertiary, and contrast CSS classes. For example:
