Buttons

We make provision for <button>, <input type="submit"> and <a role="button"> tags to be styled as a button.

CSS Classes

Individual button colors can be set using the primary, secondary, tertiary, contrast, and neutral CSS classes.

We can create button of different sizes using the small and big CSS classes.
