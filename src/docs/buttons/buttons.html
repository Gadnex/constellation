CSS Classes

There are a number of CSS classes to alter the appearence of buttons.

Colors

Individual button colors can be set using the primary, secondary, tertiary, contrast, and neutral CSS classes.

We can create button of different sizes using the sm, lg and xl CSS classes.

Ghost buttons

A ghost button is created by using the ghost CSS classes. This is useful for icon only buttons and other edge cases.
