--- title: The reel pageOrder: 10 ---

The reel is a layout component designed for making an HTML element horizontally scrollable within its parent element.

The reel is applied by setting class="reel" on an HTML element.

CSS Classes

There are a number of additional CSS classes that can be applied to a reel to alter the appearence.

Item width

There are a number CSS classes to set the width of the elements of the reel. The default item width is w-20vw, which is 20% of the viewport (screen) width.

The examples below demonstare the different gap sizes.

w-10vw
w-20vw (Default)
w-30vw
w-40vw

Gap

There are a number CSS classes to set the gap between the elements of the reel. These are gap-{0-5}, which set progressively bigger gaps as the number increases. gap-0 means no gap at all. The default gap is gap-3.

The examples below demonstare the different gap sizes.

gap-0
gap-1
gap-2
gap-3 (Default)
gap-4
gap-5

Border

The bordered CSS class created a border around the reel.

bordered

CSS Variables

The following CSS variables are defined for the reel.

--item-width
Set the width of the elements in the reel.
--gap
Set the gap size between the elements in the reel.
--padding
Set the padding size below the elements in the reel where the sidebar resides.
