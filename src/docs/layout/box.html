--- title: The box pageOrder: 2 ---

The box is a layout component designed for placing a number of HTML elements inside a bordered box. The box has a has a visible border with a border radius and some padding to ensure the content of the box does not touch the border.

The box is applied by setting class="box" on an HTML element.

CSS Variables

The following CSS variables are defined for the cluster.

--box-color - Set the font color of text in the box.
--box-background-color - Set the background color of the box.
--box-border-width - Set the box border width in px. Not suggested to use the size responsive variables here.
--box-border-color - Set the border color of the box.
--box-shadow - Set the shadow of the box. One of the shadow CSS variables could be used here.
--box-border-radius - Set the border radius of the box. It is suggested to use the --size-{1-12} CSS variables to have a responsive border radius.
--box-padding - Set the padding of the box. It is suggested to use the --size-{1-12} CSS variables to have a responsive padding size.
