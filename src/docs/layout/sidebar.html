--- title: The sidebar pageOrder: 5 ---

The sidebar is a layout component designed for creating a vertical side menu next to the main page content. There is a border line separating the sidebar and the main page content. Both the sidebar and the main page content have a padding so that their content does not touch the screen edged or the border line between them. By default the sidebar has a width of 30 characters, which should be sufficient for menu item text.

The sidebar is also responsive and on narrower screens the main page content section gets narrower. On very narrow screens where the main page content would be narrower than the sidebar, the main page content moves to below the sidebar.

By default in Constellation CSS, the HTML body pas a responsive padding, thus when using a sidebar it is suggested to set <body class="fullscreen"> to prepare the body to contain a sidebar.

The sidebar is created by wrapping both the sidebar and the main page content in an HTML tag with the class sidebar-container. On the sidebar HTML element class="sidebar" is applied.

CSS Variables

The following CSS variables are defined for the cluster.

--sidebar-background-color: Sets the background color of the sidebar.
--sidebar-border-color: Sets the border color of the sidebar conatiner and sidebar.
--sidebar-border-width: Sets the border width of the sidebar.
--sidebar-padding: Set the horizontal (inline) padding of the centered element.
