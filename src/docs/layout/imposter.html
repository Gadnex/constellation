CSS Classes

There is an additional CSS class that can be applied to an imposter to alter the appearence.

Contained

It is possible that the content of the imposter will not fit within the bounds of the parent element. In such cases the imposter will escape the bounds of the parent element. If this behaviour is unwanted, the contained utility class can be applied to the imposter.

With this utility class applied the imposted content will not escape the bounds of the parent and will have a margin around the imposter to ensure the imposter does not touch the edges of it's parent. A scollbar for the imposter will appear to enable viewing all of the content.

CSS Variables

The following CSS variables are defined for the imposter.

--positioning:
Set the positioning of the imposter element. The default positioning is absolute, but can be set to fixed if the imposted should be centered within the viewport instead of its parent element.
--margin:
Set the margin around the element so that the imposter does not touch its parent's border. A padding on the parent element does not work with an imposter. The --margin variable is only used if the utility class contained is applied.
