The frame is a layout component designed for displaying image or video content the content within an HTML element. In sets a specific aspect ratio for the frame with a defaul value of 16 / 9.

The frame is applied by setting class="frame" on an HTML element.

CSS Variables

The following CSS variables are defined for the cluster.

--aspect-ratio:
Set the aspect ratio of the frame. It is recommended to use one of the CSS variables below as a value.
--ratio-ultrawide:
Sets an aspect ratio of 18/5.
--ratio-widescreen:
Sets an aspect ratio of 16/9. This is the default value.
--ratio-golden
Sets an aspect ratio of 1.6180/1.
--ratio-landscape:
Sets an aspect ratio of 4/3.
--ratio-square:
Sets an aspect ratio of 1/1.
--ratio-portrait:
Sets an aspect ratio of 3/4.
