The container is a layout component that is applied to an HTML element that creates specific behaviours of the container element and its decendents, depending on the width of the container.

The specific width is 60ch. Some bahaviours occur when the width is more than 60ch and other behaviours occur when the width is less than 60ch.

CSS Classes

There are a number of additional CSS classes that can be applied to a container to alter the appearence and behaviour.

Mobile show and hide

There are a number CSS classes to show or hide content inside a container on mobile screens.

mobile-show
Only shown on mobile
mobile-hide
Hidden on mobile

Mobile grow and no-grow

There are a number CSS classes to grow or not grow flecbox content inside a container on mobile screens.

mobile-grow
Grown on mobile
mobile-no-grow
Not grown on mobile
