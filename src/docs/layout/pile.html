--- title: The pile pageOrder: 14 ---

The pile is a layout component designed for displaying image or video content the content within an HTML element. In sets a specific aspect ratio for the pile with a defaul value of 16 / 9.

The pile is applied by setting class="pile" on an HTML element.

Image content

If you want to center the content of an element both vertically and horizontally you can apply the CSS classes pile and centered to the element.

Video content

If you want to center the content of an element both vertically and horizontally you can apply the CSS classes pile and centered to the element.
