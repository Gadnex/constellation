The cluster is a layout component designed for placing a number of HTML elements side by side horizontally. If there is insufficient space to fit all the elements side by side, they start wrappingto the next line.

The cluster is applied by setting class="cluster" on an HTML element.

CSS Classes

There are a number of additional CSS classes that can be applied to a cluster to alter the appearence and behaviour.

Gap

There are a number CSS classes to set the gap between the elements of the cluster. These are gap-{0-5}, which set progressively bigger gaps as the number increases. gap-0 means no gap at all. The default gap is the same as gap-3.

There are also gap-r-{0-5} and gap-c-{0-5} utility classes that set only the row or column gap.

Nesting clusters

It is also possible to nest a cluster within another cluster to create useful layouts, but it is not recommended to nest more than 2 levels deep.

The below example has a cluster with the CSS classes cluster and h-space-between. Within the cluster are two more clusters with the CSS classes cluster and h-center.

When the outer cluster is not wrapped the h-center of the inner 2 clusters have no effect as the inner clusters are the size of their contents. But when the outer cluster is wrapped a CSS container query will set the CSS property flex-grow: 1; on the inner clusters so that their h-center will have an effect.
