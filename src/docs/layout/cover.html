The cover is a layout component designed for vertically centering the contents within an HTML element.

It has been decided not to implement the cover as the stack with utility class v-center can achieve the same layout. Please note that you still need to set the height of the stack.

The cover is applied by setting class="stack v-center" on an HTML element.

Horizontally centered

If you want to center the content of an element both vertically and horizontally you can apply the CSS classes stack, v-center and h-center to the element.
