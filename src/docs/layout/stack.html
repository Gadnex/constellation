--- title: The stack pageOrder: 1 ---

The stack is a layout component designed for placing a number of HTML elements vertically with a consistent gap in between each element of the stack. The margin of all direct children of the stack is removed to ensure the consistent gap between the stacked elements.

The stack is applied by setting class="stack" on an HTML element.

CSS Classes

There are a number of additional CSS classes that can be applied to a stack to alter the appearence.

Gap

There are a number CSS classes to set the gap between the elements of the stack. These are gap-{0-5}, which set progressively bigger gaps as the number increases. gap-0 means no gap at all. The default gap is gap-3.
