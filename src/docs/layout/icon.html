The icon is a layout component used for displaying icons.

If an icon is displayed alongside text in a parent element, the icon and text are properly vertically alinged with a gap between them.

Heading

CSS Classes

There are a number of additional CSS classes that can be applied to an icon to alter the appearence.

Gap

There are a number CSS classes to set the gap between the icon and its sibling elements. These are gap-{0-5}, which set progressively bigger gaps as the number increases. gap-0 means no gap at all. The default gap is gap-3.

The gap classes are applied to the parent elememt of the icon and not the ison itself.

The examples below demonstare the different gap sizes.

gap-0
gap-1
gap-2
gap-3 (Default)
gap-4
gap-5

CSS Variables

The following CSS variables are defined for the parent element of the icon.

--gap
Set the gap between the icon and its sibling elements. It is suggested to use the --size-{1-12} CSS variables to have a responsive gap size.
