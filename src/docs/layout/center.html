The center is a layout component designed for horizontally centering an HTML element within its parent element. the centered element will have a maximum width to ensure text remains readable. On narrower screens the width can shrink. Even on the narrowest screens there is a padding to ensure the centered element does not touch the screen edges.

The center is applied by setting class="center" on an HTML element.

A centered element

CSS Classes

There are a number of additional CSS classes that can be applied to a center to alter the appearence.

Width

There are a number CSS classes to set the width of the centered element. The width class w-content limits the width of the centered element to be the minimum to fit its content. The width classes w-max-{1-8}0ch sets the maximum width in characters from 10ch to 80ch in increments of 10 characters. The default gap is w-max-80ch.

CSS Variables

The following CSS variables are defined for the cluster.

--center-padding: Set the horizontal (inline) padding of the centered element.
--center-max-inline-size: Set the maximum inline size of the inline element.
