The switcher is a layout component to switch between two layout options. Either the inner elements are layed out side by side in a single row or they are layed out in a single column. This differs from the cluster in that a cluster can have multiple items on the same row, but also have multiple rows.

The switcher is applied by setting class="switcher" on an HTML element.

CSS Classes

There are a number of additional CSS classes that can be applied to a switcher to alter the appearence.

Gap

There are a number CSS classes to set the gap between the elements of the switcher. These are gap-{0-5}, which set progressively bigger gaps as the number increases. gap-0 means no gap at all. The default gap is gap-3.

CSS Variables

The following CSS variables are defined for the cluster.

--threshold: Set the horizontal (inline) size at which the switched switches. Suggested to set the value in ch units. the default threshold is 60ch.
--gap: Set the gap between the elements.
