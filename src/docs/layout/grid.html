--- title: The grid pageOrder: 8 ---

The grid is a layout component designed for placing a number of HTML elements in a grid with rows and columns. All the columns have the same width. All elements in a row have the same height, but different rows do not always have the same height.

Each column has a default minimum width of 20 characters to prevent text overflow.

On the surface it may seem similar to The cluster but it differs in a number of ways. In a cluster the size of each element is deternimed by it's content and thus elements in the same row do not have the same width. When a cluster wraps onto multiple lines, the column gaps between the different rows do not match up.

The grid is applied by setting class="grid" on an HTML element.

CSS Classes

There are a number of additional CSS classes that can be applied to a grid to alter the appearence and behaviour.

Gap

There are a number CSS classes to set the gap between the elements of the grid. These are gap-{0-5}, which set progressively bigger gaps as the number increases. gap-0 means no gap at all. The default gap is gap-3.

CSS Variables

The following CSS variables are defined for the grid.

--min-inline-size: Set the minimum inline size (width) of the grid columns. It is suggested to use ch units. Values that are too small to fit the content will create text overflow issues.
--max-cols: Set the maximum number of columns of the grid.
--gap: Set the gap between rows and columns of the grid. It is suggested to use the --size-{1-12} CSS variables to have a responsive gap size.
