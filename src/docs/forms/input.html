The HTML input element is the primary mechanism for HTML form input fields.

Input types

The input element has a type attribute and the default value is type="text", but various other types are available. The following examples show all the available types.

Input validation

When validating input fields, it is useful to show whether the entered input is valid or invalid. When doing so it is important to also make this clear for accessibility users using the aria-invalid attribute.

Constellation CSS makes use of the aria-invalid attribute to apply either an --success or --error color to mark the field visually. For invalid fields, any sibling element immediately following the input element will have smaller text in the --error color.

Input type range

The input type="range" does not visually look like a text input field and is thus styled differently.
