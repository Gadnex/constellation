--- title: Dynamic colors pageOrder: 1 ---

In order to prevent an overload of color choices, Constellation CSS provides a limited set of colors to work with. These choices are dynamically generated using color theory based on a --brand color that you set in your custom CSS.

All colors have light mode and dark modes versions to ensure proper contrast with the background color. Dark mode colors are slightly lighter for improved contrast.

Brand

The default brand color of Constellation CSS is #238fff, which is the color code for dodgerblue. You can change the brand color to your preferred brand color by settting the CSS variable --brand to a valid color value.

It is suggested not to use the --brand CSS variable in your own CSS and to rather use the --primary-3 CSS variable that will work well in light and dark mode.

At the top right orner of the site you can find a settings menu. Inside the setting menu you can pick a --brand color that will immediately be applied accross the entire site. There is also a Reset button to reset back to the default brand color.

--brand:

Primary

The primary color is derived from the brand color by narrowing the lightness to a range that will look good on a light background. Thus if you select a --brand color that is too light or dark, the primary color will fix this issue.

The primary color will have two ligher and two darker shades that you can use as CSS variables. Other colors also have 2 ligher and darker shades.

The color variables --primary-{1-5} are meant to be applied from back to front on your website and they work for both light and dark modes.

Secondary

The secondary colors are relative to the primary colors and are found by moving the hue 35 degrees in a clockwize direction on the color wheel. The value can be adjusted in the settings menu.

--secondary-1
--secondary-2
--secondary-3

Tertiary

The tertiary colors are relative to the primary colors and are found by moving the hue 35 degrees in an anti- clockwize direction on the color wheel. The value can be adjusted in the settings menu.

--tertiary-1
--tertiary-2
--tertiary-3

Contrast

The contrast colors are relative to the primary color and are found by moving the hue 180 degrees in a clockwize direction on the color wheel. The value is thus on the opposite side of the color wheel and cannot be adjusted.

--contrast-1
--contrast-2
--contrast-3

Neutral

The neutral colors are different shades of grey with a slight hint of the primary color to give them some character.

Semantic

A number of scemantic colors are also offered to highlight error, warning, success and info details. These values are not customizable and care should be taken so that the dynamically generated colors do not clash with these colors.

--error
--warning
--success
--info
