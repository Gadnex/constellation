A dialog is a modal or non-modal popup that sits on top of the other page content and it obscures the page content with a partially tranparent backdrop.

It is created using the <dialog> element.

Dialog

This dialog has class="box" applied to create the border, border radius and the padding. For dialog elements it is not required to add the bordered class to the box as dialogs always have a border.

To open the dialog the commandFor="dialog-id" and command="show-modal" attributes were put on the Show Dialog button. This opens the dialog without JavaScript as a modal dialog.

To close the dialog the close button or the escape key can be used.
