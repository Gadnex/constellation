Constellation CSS provides fluent typography styles for headings and other text, ensuring that your content is both visually appealing and easy to read. The typography system is designed to be flexible, allowing you to customize the appearance of your text while maintaining a consistent look and feel across your website.

CSS utility classes

A number of CSS classes are available to apply text colors to your content. These classes allow you to easily change the color of your text to match your design and branding, while maintaining a consistent look and feel across your website.

Text colors are inherited by default, so you can apply these classes to any parent element to change the color of all child text elements. This makes it easy to create sections of your website with different color schemes without having to apply classes to each individual text element.

text-default

Default paragraph text applied using class text-default on a div element surrounding the heading and paragraph.

text-primary

Primary paragraph text applied using class text-primary on a div element surrounding the heading and paragraph.

This paragraph is also within the div, but if has class text-default applied to reset the color to the default.

text-secondary

Secondary paragraph text applied using class text-secondary on a div element surrounding the heading and paragraph.

text-tertiary

Tertiary paragraph text applied using class text-tertiary on a div element surrounding the heading and paragraph.

text-contrast

Contrast paragraph text applied using class text-contrast on a div element surrounding the heading and paragraph.

text-error

Error paragraph text applied using class text-error on a div element surrounding the heading and paragraph.

text-warning

Warning paragraph text applied using class text-warning on a div element surrounding the heading and paragraph.

text-success

Success paragraph text applied using class text-success on a div element surrounding the heading and paragraph.

text-info

Info paragraph text applied using class text-info on a div element surrounding the heading and paragraph.
