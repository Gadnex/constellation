HTML has a number of list elements. The most well known list elements are ordered lists and unordered lists, but these are not the only variants.

Ordered lists

Ordered lists have items numbered from one onwards. They are created using the ol element with nested li elements.

  1. Item one
  2. Item two
  3. Item three

Unordered lists

Unordered lists have items presented as a bulleted list. They are created using the ul element with nested li elements.

  • Item one
  • Item two
  • Item three

Menu lists

HTML also has a menu element that is treated by the browser as an ul. They are created using the menu element with nested li elements.

Constellation CSS has made an opinionated decition to remove the bullets for a menu. Anchor a elements inside the list items are also converted to block level elements so that they fill the entire width of their parent. This has the effect that you can click next to the anchor to navigate the link.

You can also put the aria-current="page" attibute on the anchor element to show that this is the current page in the menu.

  • Menu item one
  • Menu item two
  • Menu item three
